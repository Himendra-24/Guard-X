## 💡 About The Project

Guard-X is a **personal safety application** that helps users during emergencies by automatically triggering alerts when the user is unable to respond.

It focuses on:
- ⚡ Fast emergency response  
- 📡 Offline functionality  
- 🤖 Automated safety logic  

---

## ✨ Key Features

### 🚨 Smart SOS System
- Instant emergency alerts to contacts  

### ⏱ Trip Monitoring
- Start trip with timer  
- Continuous safety tracking  

### ⚠️ Auto Alert System
- No response → Alarm + alert flow  

### 📩 Offline Location Sharing
- SMS-based location sharing (no internet required)

### 😏 Fake Call Feature
- Escape from uncomfortable situations  

### 🔦 Emergency Toolkit
- Siren 🔊  
- Flashlight 🔦  
- Quick actions  

### 📊 Activity Logs
- Track trips and alerts  

### 🔄 Firebase Sync
- Sync data across sessions  

---

## 🧠 Tech Stack

- Flutter  
- Hive (Local Storage)  
- Firebase (Auth & Sync)  
- Notifications  
- Device Features (SMS, Location, Flashlight, Calls)
